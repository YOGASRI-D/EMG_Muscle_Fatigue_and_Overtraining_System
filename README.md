# EMG_Muscle_Fatigue_and_Overtraining_System
ESP32-based EMG muscle fatigue and overtraining monitoring system with real-time web visualization.

## Overview
This project captures EMG signals using ESP32 and visualizes them on a web dashboard for real-time monitoring and muscle fatigue analysis.

## Features
- Real-time EMG signal acquisition
- Live data visualization
- Muscle fatigue detection
- Web-based monitoring interface

## Tech Stack
- ESP32 (Arduino)
- MERN Stack (MongoDB, Express, React, Node.js)

## System Architecture
1. EMG sensor acquires muscle signals  
2. ESP32 reads and sends data via WiFi  
3. Backend server receives and processes data  
4. Frontend dashboard displays live charts  

## Future Scope
- AI-based fatigue prediction
- Signal filtering and advanced analysis
